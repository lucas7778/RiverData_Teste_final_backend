# RiverData Case

## Rodar o back
*Adicionar no arquivo .env o endereço do endpoint do llama 2 e o token.*

```bash
cd back
npm install
npm start
```
## Estrutura do back
Foi desenvolvido com clean architecture e express

## Rodar o front
```bash
cd front
npm install
npm start
```
## Estrutura front
Foi desenvolvido em react com vite

## Entrar na aplicação
abrir https://localhost:8000
